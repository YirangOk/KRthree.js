# KRthree.js

한국어 함수로 three.js 3D 코딩을 하는 라이브러리입니다. **three.js가 안에 들어 있어** 따로 설치할 필요가 없습니다.

```js
삼차원시작 → Scene + Camera + Renderer   |   큐브만들기 → BoxGeometry + Mesh   |   책만들기 → 책 모양
```

## 들어 있는 파일

```
KRthree.js/
├─ KRthree.js     ← 라이브러리 한 개 (three.js 포함, 이 파일만 있으면 됩니다)
├─ index.html     ← 그대로 두세요
└─ 내코드.js        ← 여기를 고치세요 (랜덤 색 책장 예제)
```

## VS Code로 쓰기 (3단계)

1. **VS Code** 설치 → 확장에서 **Live Server** 설치
2. 이 폴더를 VS Code로 열기
3. `index.html` 우클릭 → **"Open with Live Server"** → 랜덤 색 책장이 돌아갑니다.

`내코드.js` 를 고치고 저장하면 브라우저가 자동으로 새로고침됩니다.
미리보기를 **마우스로 드래그**하면 책장 주위를 돌아다닐 수 있어요.

## 새 프로젝트에 넣기

`KRthree.js` 파일을 복사하고 HTML에서 이렇게 불러옵니다.

```html
<script src="KRthree.js"></script>
<script>
  // 한국어 함수를 전역으로 등록 (이 두 줄은 그대로)
  Object.assign(window, KCC3D)
  window.THREE = KCC3D.THREE
</script>
<script>
  const { 장면, 카메라, 렌더러 } = 삼차원시작({ 배경: '#1a1a1a' })
  const 큐브 = 큐브만들기(2, { 색: '하늘색' })
  장면에추가하기(장면, 큐브)
  애니메이션시작(() => {
    회전하기(큐브, 0.01, 0.01)
    렌더링하기(렌더러, 장면, 카메라)
  })
</script>
```

> three.js 원본이 필요하면 `window.THREE` 로 그대로 쓸 수 있습니다.

## 자주 쓰는 함수

| 한국어 | 하는 일 |
| --- | --- |
| `삼차원시작({ 배경 })` | 장면·카메라·렌더러를 한 번에 |
| `위치설정 · 바라보기` | 위치 정하기 / 그 점을 바라보게 돌리기 |
| `빛추가하기(장면, 종류, 옵션)` | 빛 켜기 (주변빛 / 방향빛 / 점광 / 스포트라이트) |
| `큐브만들기 · 직육면체만들기 · 책만들기 · 구만들기` | 도형·책 만들기 |
| `재질만들기 · 텍스처불러오기` | 재질·텍스처 |
| `그룹만들기 · 그룹에추가하기` | 여러 객체 묶기 |
| `회전하기 · 시계만들기 · 경과시간` | 애니메이션 |
| `장면에추가하기` | 장면에 올리기(올려야 보임) |
| `애니메이션시작 · 렌더링하기` | 매 프레임 반복 / 화면에 그리기 |

색 이름은 `흰색 검정 빨강 파랑 초록 노랑 하늘색 …` 같은 한국어나 `'#e74c3c'` 색 코드를 씁니다.
